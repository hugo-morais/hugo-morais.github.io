# Personal Portfolio Website

This repository contains my personal portfolio website.

The site showcases some of my projects related to data, software development, and DevOps.

## Features

- Project showcase with images and descriptions
- Links to GitHub repositories
- Links to Linkedin
- Interactive project cards


## Projects Featured

Some projects displayed on the site include:

- **DevOps User API** – REST API with Docker containerization, CI pipeline, MariaDB database, and Kubernetes orchestration
- **Student Grades ETL** – Mini ETL pipeline that processes student grades and exports structured data to JSON
- **Stock Management System** – Command-line application in C++ for managing store inventory

## Deployment

The website is deployed using **GitHub Pages**.

## Purpose

This project serves as a simple portfolio to present my personal projects and experiments.
